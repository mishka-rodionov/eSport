# Sports Enthusiast (eSport)

**Sports Enthusiast** — это современное Android-приложение для управления спортивными соревнованиями, ориентированное на организаторов и участников соревнований по спортивному ориентированию. Приложение позволяет управлять списками участников, проводить жеребьевку, контролировать выдачу чипов и фиксировать результаты с помощью NFC.

## 🚀 Основные возможности

- **Управление соревнованиями:** Создание и настройка параметров событий.
- **Работа с участниками:** Просмотр списков, группировка по категориям и автоматическая жеребьевка.
- **NFC Интеграция:** Считывание и запись данных на чипы для ориентирования через NFC модуль.
- **Контроль выдачи чипов:** Удобный интерфейс для отметки выданного оборудования.
- **Результаты в реальном времени:** Отслеживание статуса участников и формирование итоговых протоколов.

## 🛠 Стек технологий

- **Язык:** [Kotlin](https://kotlinlang.org/)
- **UI:** [Jetpack Compose](https://developer.android.com/jetpack/compose) (Modern UI)
- **Архитектура:** Clean Architecture + MVVM + Multi-module
- **DI:** [Koin](https://insert-koin.io/)
- **Асинхронность:** Coroutines & Flow
- **Сеть:** Retrofit + OkHttp
- **База данных:** Room
- **Навигация:** Compose Navigation
- **Сборка:** Gradle (Kotlin DSL)

## 🏗 Архитектура проекта

Проект разделен на модули для обеспечения высокой масштабируемости и тестируемости:

### Схема модулей:
- `:app` — основной модуль (точка входа).
- `:domain` — бизнес-логика, модели данных и интерфейсы репозиториев.
- `:data` — реализация данных (Network, Local Room DB).
- `:feature` — функциональные модули (Center, Events, Profile).
- `:core` — общие компоненты (UI Kit, Design System, NFC Helper).
- `:utils` — вспомогательные расширения и утилиты.
